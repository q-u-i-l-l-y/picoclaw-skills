## Quick Start (Clone-to-Node)

```bash
# 1. Clone the skills library
git clone https://github.com/q-u-i-l-l-y/picoclaw-skills.git
cd picoclaw-skills

# 2. Install dependencies and register as mesh node
bash install.sh

# 3. Your agent is now running as a spoke node
#    Run status to see your node's capabilities:
python3 -m picoclaw status
```

### What Happens During Install

1. **Skill Discovery** — Scans `/skills/` directory for manifests
2. **Capability Detection** — Discovers your device's available tools
3. **Node Registration** — Generates unique node identity (pubkey-based)
4. **Mesh Join** — Registers as spoke in the collective network
5. **Revenue Engine** — Activates Tier 1 skills (affiliate pipeline)
6. **North Star Alignment** — All actions aligned with the grand vision
